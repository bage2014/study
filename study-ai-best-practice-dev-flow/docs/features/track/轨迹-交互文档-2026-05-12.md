# 轨迹-交互文档

## 1. 交互流程

### 1.1 地图加载流程

```mermaid
flowchart TD
    A[用户进入轨迹页面] --> B[加载百度地图]
    B --> C[加载用户轨迹数据]
    C --> D[在地图上渲染轨迹点]
    D --> E[绘制轨迹连线]
```

### 1.2 踩点流程

```mermaid
flowchart TD
    A[用户点击地图] --> B[获取点击位置坐标]
    B --> C[弹出位置信息弹窗]
    C --> D[用户输入位置名称和描述]
    D --> E[点击保存]
    E --> F[发送保存请求]
    F --> G[更新地图显示]
```

### 1.3 轨迹显示流程

```mermaid
flowchart TD
    A[轨迹数据加载完成] --> B[按时间排序轨迹点]
    B --> C[创建地图标记点]
    C --> D[创建Polyline连接点]
    D --> E[渲染到地图]
```

## 3. 界面设计

### 3.1 轨迹页面布局

| 元素 | 描述 | 位置 |
|------|------|------|
| 标题 | "用户轨迹" | 页面顶部 |
| 操作按钮 | 清空轨迹、导出轨迹 | 标题右侧 |
| 地图容器 | 百度地图显示区域 | 页面主体 |
| 轨迹列表 | 轨迹点列表（侧边栏） | 页面右侧 |

### 3.2 踩点弹窗

| 元素 | 描述 | 位置 |
|------|------|------|
| 标题 | "添加轨迹点" | 弹窗顶部 |
| 名称输入框 | 文本输入，placeholder:"位置名称" | 表单第一行 |
| 描述输入框 | 文本域，placeholder:"位置描述（可选）" | 表单第二行 |
| 坐标显示 | 显示点击的经纬度 | 表单第三行 |
| 确定按钮 | 蓝色主按钮 | 弹窗底部左侧 |
| 取消按钮 | 白色按钮 | 弹窗底部右侧 |

### 3.3 轨迹点标记

| 元素 | 描述 | 样式 |
|------|------|------|
| 标记图标 | 蓝色圆形标记 | 半径10px，蓝色边框 |
| 序号标记 | 显示轨迹点顺序 | 标记中心数字 |
| 连线 | 连接相邻轨迹点 | 蓝色实线，宽度3px |
