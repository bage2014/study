### 1.2 组件划分

| 组件 | 说明 | 状态 |
|------|------|------|
| PredictionButton | 关系预测触发按钮 | 可点击 |
| PredictionResult | 预测结果卡片 | 展示 |
| StorySelector | 故事类型选择器 | 可选择 |
| StoryResult | 故事内容展示 | 展示 |

## 二、交互流程

### 2.1 关系预测流程

```mermaid
graph TD
    A[点击开始预测] --> B[显示加载动画]
    B --> C[AI分析家族数据]
    C --> D[展示预测结果]
    D --> E{用户确认?}
    E -->|是| F[建立关系]
    E -->|否| G[继续分析]
```

### 2.2 故事生成流程

```mermaid
graph TD
    A[选择故事类型] --> B[点击生成故事]
    B --> C[显示加载动画]
    C --> D[AI生成故事]
    D --> E[展示故事内容]
    E --> F[导出分享]
```

## 三、界面原型

### 3.1 关系预测结果

**布局：**
- 卡片式展示
- 圆角8px
- 渐变背景

**元素样式：**
| 元素 | 样式 |
|------|------|
| 成员名称 | 16px，粗体 |
| 关系类型 | 背景色标签 |
| 置信度 | 绿色高亮 |
| 推理依据 | 灰色小字 |

### 3.2 故事生成结果

**布局：**
- 卡片式展示
- 白色背景
- 文字居中

**元素样式：**
| 元素 | 样式 |
|------|------|
| 标题 | 20px，粗体 |
| 内容 | 16px，行高1.8 |
| AI点评 | 背景色块 |

## 四、状态说明

| 状态 | 描述 | 界面表现 |
|------|------|----------|
| 初始状态 | 页面加载完成 | 显示触发按钮 |
| 加载中 | AI分析中 | 显示加载动画 |
| 完成状态 | 分析完成 | 显示结果 |
| 空状态 | 无数据 | 提示添加数据 |
