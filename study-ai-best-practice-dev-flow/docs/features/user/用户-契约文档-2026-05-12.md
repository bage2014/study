# 用户-契约文档-2026-05-12

## 1. API 契约概述

用户管理模块 API 接口定义。

## 2. 接口列表

### 2.1 获取用户列表

| 属性 | 值 |
|------|-----|
| 路径 | GET /api/users |
| 方法 | GET |

### 2.2 获取单个用户

| 属性 | 值 |
|------|-----|
| 路径 | GET /api/users/{id} |
| 方法 | GET |

### 2.3 创建用户

| 属性 | 值 |
|------|-----|
| 路径 | POST /api/users |
| 方法 | POST |

### 2.4 更新用户

| 属性 | 值 |
|------|-----|
| 路径 | PUT /api/users/{id} |
| 方法 | PUT |

### 2.5 删除用户

| 属性 | 值 |
|------|-----|
| 路径 | DELETE /api/users/{id} |
| 方法 | DELETE |

```json
[

    "id": 1,
```
