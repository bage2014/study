# /design 命令
# 技术方案与评审命令

## 功能描述
生成技术方案设计并进行自动评审。

## 自动执行的任务序列

1. **数据库设计**
   - 设计实体模型
   - 定义表结构
   - 确定关系映射

2. **API 定义**
   - 定义 REST API 接口
   - 设计请求/响应格式
   - 定义错误码

3. **技术选型**
   - 确定使用的框架和库
   - 定义项目结构
   - 确定编码规范

4. **安全评审**
   - 检查 SQL 注入风险
   - 检查 XSS 风险
   - 检查敏感信息暴露

5. **性能评审**
   - 检查数据库查询效率
   - 检查 API 响应时间
   - 检查前端渲染性能

6. **输出评审意见**
   - 生成设计文档
   - 输出评审报告
   - 提出改进建议

## 使用方式

```
/design [功能模块]
```

## 示例

```
/design 用户登录模块
/design 用户轨迹模块
```

## 输出

- 设计文档：`docs/design/DESIGN_[模块]_v[版本].md`
- 包含内容：
  - 数据库 ER 图
  - API 接口列表
  - 数据模型
  - 安全和性能评审意见

## 命名规范

设计文档命名：`DESIGN_[功能简写]_v[版本].md`
- 例如：`DESIGN_login_v1.0.md`、`DESIGN_track_v1.0.md`
