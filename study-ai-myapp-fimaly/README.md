# 家庭族谱APP

## 项目概述

本项目旨在实现一个完整的家庭族谱APP应用，包括Web端、Android端和后端服务。通过该应用，用户可以创建、管理和查看家族成员信息，构建完整的家族树，记录家族历史和重要事件。

## 技术栈

- **后端**：Spring Boot
- **Web前端**：现代化Web技术栈
- **Android端**：原生Android开发
- **数据存储**：关系型数据库

## 项目结构

```
├── backend/           # 后端Spring Boot项目
├── frontend/          # 前端项目
│   ├── web/           # Web端实现
│   └── android/       # Android端实现
├── docs/              # 项目文档
│   ├── prd/               # 产品需求文档 + 产品调研文档
│   ├── ui-design/         # UI设计文档
└── README.md          # 项目说明文档
```

## 主要功能

1. **家族成员管理**：添加、编辑、删除家族成员信息
2. **家族树可视化**：以树形结构展示家族关系
3. **家族历史记录**：记录家族重要事件和历史
4. **多媒体资料管理**：上传和管理家族照片、视频等资料
5. **用户认证与授权**：确保数据安全
6. **数据备份与恢复**：保障数据安全
