# 家庭族谱APP用户指南

## 1. 系统概述

家庭族谱APP是一个功能完整的家族管理系统，帮助用户创建、管理和查看家族成员信息，构建完整的家族树，记录家族历史和重要事件。

### 1.1 主要功能
- **家族成员管理**：添加、编辑、删除家族成员信息
- **家族树可视化**：以树形结构展示家族关系
- **家族历史记录**：记录家族重要事件和历史
- **多媒体资料管理**：上传和管理家族照片、视频等资料
- **用户认证与授权**：确保数据安全
- **数据备份与恢复**：保障数据安全

## 2. 注册与登录

### 2.1 注册账号
1. 访问系统首页，点击"注册"按钮
2. 填写注册信息：
   - 邮箱地址
   - 密码（至少8位，包含字母和数字）
   - 昵称
   - 手机号（可选）
3. 点击"注册"按钮完成注册

### 2.2 登录系统
1. 访问系统首页，点击"登录"按钮
2. 输入注册时使用的邮箱和密码
3. 点击"登录"按钮进入系统

### 2.3 忘记密码
1. 在登录页面点击"忘记密码"
2. 输入注册时使用的邮箱
3. 按照邮件提示重置密码

## 3. 家族管理

### 3.1 创建家族
1. 登录系统后，点击左侧菜单"家族管理"
2. 点击"创建家族"按钮
3. 填写家族信息：
   - 家族名称
   - 家族描述
4. 点击"保存"按钮完成创建

### 3.2 管理家族成员
1. 选择一个家族
2. 点击"成员管理"选项
3. 可以进行以下操作：
   - **添加成员**：点击"添加成员"按钮，填写成员信息
   - **编辑成员**：点击成员信息旁的"编辑"按钮
   - **删除成员**：点击成员信息旁的"删除"按钮
   - **查看详情**：点击成员姓名查看详细信息

### 3.3 家族树查看
1. 选择一个家族
2. 点击"家族树"选项
3. 系统会以树形结构展示家族成员关系
4. 可以通过鼠标滚轮缩放查看
5. 点击成员节点查看详细信息

## 4. 成员管理

### 4.1 成员信息管理
1. 点击左侧菜单"成员管理"
2. 可以按照姓名、性别、出生日期等条件搜索成员
3. 点击成员姓名进入详情页
4. 在详情页可以：
   - 编辑基本信息
   - 添加家庭成员关系
   - 上传相关照片和视频
   - 记录重要事件和里程碑

### 4.2 成员关系管理
1. 在成员详情页，点击"关系管理"
2. 点击"添加关系"按钮
3. 选择相关成员和关系类型（如父子、配偶等）
4. 点击"保存"按钮

## 5. 事件管理

### 5.1 创建事件
1. 点击左侧菜单"事件管理"
2. 点击"创建事件"按钮
3. 填写事件信息：
   - 事件标题
   - 事件描述
   - 事件日期
   - 事件地点
   - 相关家族
4. 点击"保存"按钮

### 5.2 事件详情
1. 点击事件列表中的事件标题
2. 在详情页可以：
   - 编辑事件信息
   - 添加相关照片和视频
   - 关联相关成员

## 6. 媒体管理

### 6.1 上传媒体
1. 点击左侧菜单"媒体管理"
2. 点击"上传媒体"按钮
3. 选择文件类型（照片/视频）
4. 选择要上传的文件
5. 填写媒体信息：
   - 标题
   - 描述
   - 相关家族
   - 相关成员（可选）
   - 相关事件（可选）
6. 点击"上传"按钮

### 6.2 媒体管理
1. 在媒体列表中可以：
   - 查看媒体预览
   - 编辑媒体信息
   - 删除媒体文件
   - 下载媒体文件

## 7. 家族故事

### 7.1 创建家族故事
1. 点击左侧菜单"家族故事"
2. 点击"创建故事"按钮
3. 填写故事信息：
   - 标题
   - 内容
   - 相关家族
   - 相关成员
   - 发布日期
4. 点击"保存"按钮

### 7.2 故事管理
1. 在故事列表中可以：
   - 查看故事详情
   - 编辑故事内容
   - 删除故事

## 8. 地点管理

### 8.1 添加地点
1. 点击左侧菜单"地点管理"
2. 点击"添加地点"按钮
3. 填写地点信息：
   - 地点名称
   - 经度
   - 纬度
   - 描述
4. 点击"保存"按钮

### 8.2 地点地图
1. 点击"地点地图"选项
2. 系统会在地图上显示所有已添加的地点
3. 点击地图上的标记查看地点详情

## 9. 系统设置

### 9.1 个人设置
1. 点击右上角用户头像，选择"个人设置"
2. 可以修改：
   - 昵称
   - 密码
   - 联系方式

### 9.2 家族设置
1. 选择一个家族，点击"家族设置"
2. 可以修改：
   - 家族名称和描述
   - 家族管理员
   - 成员权限

### 9.3 数据备份
1. 点击左侧菜单"系统设置"
2. 点击"数据备份"
3. 选择备份范围
4. 点击"开始备份"按钮
5. 系统会生成备份文件并提供下载

## 10. 权限管理

### 10.1 角色权限
- **管理员**：拥有所有权限，包括家族管理、成员管理、事件管理等
- **成员**：可以查看和编辑自己家族的信息
- **访客**：只能查看公开信息

### 10.2 权限设置
1. 点击左侧菜单"权限管理"
2. 选择一个家族
3. 点击"添加权限"按钮
4. 选择用户和角色
5. 点击"保存"按钮

## 11. 常见问题

### 11.1 如何添加家族成员？
1. 进入家族管理页面
2. 选择要添加成员的家族
3. 点击"成员管理"选项
4. 点击"添加成员"按钮
5. 填写成员信息并保存

### 11.2 如何查看家族树？
1. 进入家族管理页面
2. 选择要查看的家族
3. 点击"家族树"选项
4. 系统会自动生成并显示家族树

### 11.3 如何上传家族照片？
1. 点击左侧菜单"媒体管理"
2. 点击"上传媒体"按钮
3. 选择照片文件
4. 填写相关信息并上传

### 11.4 如何修改密码？
1. 点击右上角用户头像
2. 选择"个人设置"
3. 点击"修改密码"
4. 输入旧密码和新密码
5. 点击"保存"按钮
