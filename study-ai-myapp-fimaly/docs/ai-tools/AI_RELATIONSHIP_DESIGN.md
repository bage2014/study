- 年龄差18-40岁 → 可能亲子关系
- 年龄差≤10岁 → 可能兄弟姐妹关系


## 三、核心功能模块设计

### 3.1 智能关系预测（Smart Relationship Prediction）

#### 3.1.1 多维度分析引擎

| 分析维度 | 输入数据 | 分析方法 | 输出结果 |
|---------|---------|---------|---------|
| **年龄分析** | 出生日期 | 年龄差计算 | 亲子/兄弟姐妹/祖孙关系 |
| **姓名分析** | 姓名 | 姓氏溯源/字辈分析 | 同族可能性 |
| **地域分析** | 籍贯/居住地 | 地理距离计算 | 迁移路径/同乡关系 |
| **照片分析** | 头像/全家福 | 人脸识别+相似度 | 基因相似度评估 |
| **特征分析** | 体型/外貌描述 | 特征匹配 | 遗传特征关联 |
| **职业分析** | 工作/职位 | 行业关联分析 | 可能的社会关系 |
| **事件分析** | 家族事件记录 | 时间线关联 | 事件参与者关系 |

### 3.2 家族故事生成（Family Story Generation）

#### 3.2.1 故事类型

| 故事类型 | 生成内容 | 目标用户 | 价值 |
|---------|---------|---------|------|
| **家族迁徙史** | 家族迁移路线、时间、原因 | 所有用户 | 文化传承 |
| **人物传记** | 家族成员生平事迹 | 后代子孙 | 精神遗产 |
| **家族节日** | 传统节日习俗、礼仪 | 家族成员 | 文化传承 |
| **家训家规** | 家族规矩、价值观 | 教育后代 | 品德培养 |
| **传奇故事** | 祖辈传说、轶事 | 增添趣味 | 家族认同 |
| **家族变迁** | 家族兴衰史 | 历史记录 | 经验借鉴 |


### 3.3 人脸识别匹配（Face Recognition Matching）

#### 3.3.1 功能设计

| 功能 | 描述 | 技术方案 |
|------|------|---------|
| **家庭相册人脸聚类** | 自动识别同一人物的不同照片 | face-api.js + 聚类算法 |
| **跨代相貌遗传分析** | 分析祖孙/父子相似度 | 人脸特征向量比较 |
| **疑似亲属人脸匹配** | 在家族库中匹配相似人脸 | 人脸检索 + 向量数据库 |
| **证件照自动关联** | 将证件照与家庭成员关联 | 人脸1:1比对 |

```java
// AI服务门面 - 统一入口
public interface AiServiceFacade {
    // 关系预测

    // 故事生成
    FamilyStory generateStory(StoryRequest request);
    
    // 人脸匹配
    FaceMatchResult matchFace(FaceMatchRequest request);
    
    // 数据补全
    CompletionSuggestion suggestCompletion(SuggestRequest request);
}

// 多Provider支持
public interface AiProvider {
    String getProviderName();
    boolean isAvailable();
    AiResponse generate(Prompt prompt);
    double getCostPerToken();
}
```
