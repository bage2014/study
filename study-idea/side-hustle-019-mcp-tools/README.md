# 副业Idea：MCP工具开发

## Idea概述

开发MCP（Model Context Protocol）插件和工具，帮助AI模型连接外部数据和服务，构建AI生态基础设施。

## 核心价值

- 新兴生态，先发优势明显
- 技术门槛适中，适合技术人
- 产品化程度高，可规模化

- AI开发者
- AI产品公司
- 企业技术团队
- AI平台服务商
