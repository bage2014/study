## 1. 位置类规律

位置类规律是图形推理中最基础的题型之一，考查的是图形元素在空间中的位置变化关系。

### 1.1 平移规律

#### 知识点与考点

**核心概念**：图形中的某个元素按照固定方向和距离移动。

**移动方向**：
- 水平移动（左→右 或 右→左）
- 垂直移动（上→下 或 下→上）
- 对角线移动（如右上→左下）
- 环形移动（绕边界循环）

**移动距离**：
- 固定步数（每次移动1格、2格等）
- 递增步数（每次移动距离增加）
- 递减步数（每次移动距离减少）

### 1.2 旋转规律

#### 知识点与考点

**核心概念**：图形绕某点旋转一定角度。

**旋转要素**：
- 旋转中心：图形中心、顶点、边中点
- 旋转方向：顺时针、逆时针
- 旋转角度：45°、90°、180°等

---

#### 题目样例（10道）

**【例1】90°顺时针旋转**

```
题目：观察下列图形序列，找出规律并选择正确答案。

│L│    │─│    │Γ│    │││    │L│
│ │    │  │    │ │    │  │    │ │
```

## 4. 属性类规律

属性类规律是图形推理中的重要题型，考查的是图形的属性特征。

### 4.1 对称性

#### 知识点与考点

**核心概念**：图形的对称特征。

**对称类型**：
- 轴对称：图形沿某条直线对称
- 中心对称：图形绕某点旋转180°后与原图形重合
- 既轴对称又中心对称：图形同时具有轴对称和中心对称特征

---

#### 题目样例（10道）

```
  △      ▽      ◇      □      ?

对称性：轴对称  轴对称  轴对称  轴对称  ?

规律分析：所有图形均为轴对称图形。

选项：
A. 轴对称图形    B. 中心对称图形    C. 既轴对称又中心对称    D. 不对称图形
```

1. **位置类规律**：观察图形元素的位置变化，包括平移、旋转、翻转等。

3. **样式类规律**：观察图形元素的样式变化，包括遍历、运算等。
4. **属性类规律**：观察图形的属性特征，包括对称性、曲直性、封闭性等。
5. **空间重构类**：观察图形的空间重构，包括六面体、四面体、多面体等。

**解题步骤**：
1. 观察图形序列，找出规律。

4. 验证答案，确保正确。
