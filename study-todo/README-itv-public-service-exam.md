## 1. 行政职业能力测验 (行测)

### 1.1 言语理解与表达

#### 1.1.1 主旨概括题

**解题技巧**：
- **找主题句**：通常在段首或段尾
- **关联词法**：转折、因果、递进关系
- **高频词法**：重复出现的关键词

**示例**：
```
题目：随着科技的发展，人工智能已经渗透到我们生活的方方面面。从智能家居到自动驾驶，从医疗诊断到金融分析，AI正在改变着我们的生活方式。

这段文字的主旨是：
A. 科技发展的重要性
B. AI对生活的影响
C. 智能家居的普及
D. 人工智能的定义

答案：B
```

#### 1.1.2 意图推断题

**解题技巧**：
- **把握作者态度**：积极、消极、中立
- **推断言外之意**：不要局限于字面意思
- **排除字面表述**：正确选项通常是引申义

#### 1.1.3 细节判断题

**解题技巧**：
- **逐句比对**：与原文逐一核对
- **注意偷换概念**：时态、数量、范围
- **警惕绝对化表述**：所有、都、一定

#### 1.1.4 语句排序题

**解题技巧**：
- **找首尾句**：总起句或总结句
- **找连接词**：转折、因果、递进
- **找话题一致**：相同主语或话题

### 1.2 数量关系

#### 1.2.1 数字推理

**常见数列类型**：

| 类型 | 特点 | 示例 |
|------|------|------|
| 等差数列 | 相邻两项差相等 | 1, 3, 5, 7, 9... |
| 等比数列 | 相邻两项比相等 | 2, 6, 18, 54... |
| 递推数列 | 前项运算得到后项 | 1, 2, 3, 5, 8... |
| 幂次数列 | 平方、立方关系 | 1, 4, 9, 16, 25... |
| 组合数列 | 多个规律组合 | 1, 1, 2, 4, 3, 9... |

**解题技巧**：
1. 观察数列变化趋势
2. 尝试作差、作商
3. 考虑幂次、递推关系

#### 1.2.2 数学运算

**高频考点**：

1. **行程问题**：
   - 相遇问题：路程 = 速度和 × 时间
   - 追及问题：路程差 = 速度差 × 时间
   - 流水行船：顺水速度 = 船速 + 水速

2. **工程问题**：
   - 工作量 = 效率 × 时间
   - 多人合作：总效率 = 各效率之和

3. **利润问题**：
   - 利润 = 售价 - 成本
   - 利润率 = 利润 ÷ 成本 × 100%

4. **排列组合**：
   - 排列：P(n,m) = n! / (n-m)!
   - 组合：C(n,m) = n! / [m!(n-m)!]

5. **概率问题**：
   - 古典概率：P(A) = 符合条件的情况数 / 总情况数
   - 条件概率：P(A|B) = P(AB) / P(B)

**速算技巧**：
- 尾数法：利用尾数判断答案
- 代入法：将选项代入验证
- 特殊值法：用特殊值简化计算

### 1.3 判断推理

#### 1.3.1 图形推理

**常见规律**：

| 类型 | 规律 | 示例 |
|------|------|------|
| 位置类 | 平移、旋转、翻转 | 图形元素位置变化 |
| 数量类 | 点、线、角、面、素 | 数量递增或递减 |
| 样式类 | 遍历、运算、属性 | 图形叠加、求同存异 |
| 空间类 | 折叠、三视图 | 立方体展开图 |

**解题技巧**：
1. 观察图形特征
2. 尝试常见规律
3. 验证规律是否适用

#### 数量类——点、线、面、角、素

**一、点**

**规律说明**：图形中交点的数量存在递增、递减或不变等规律。常见的点包括：线条交点、线条与封闭空间的交点等。

**题目样例**：

**【例1】**
```
题目：下列选项中，符合图形规律的是（ ）。

选项：A. 2个点  B. 3个点  C. 4个点  D. 5个点

答案：C
解析：图形中交点数量依次为1、2、3、4，故下一个应为5个点，但选项中无5，故选4个点对应的图形。
```

**【例2】**
```
题目：每行图形中，交点数量相同的是（ ）。

选项：A. 第一行  B. 第二行  C. 第三行  D. 三行都不相同

答案：B
解析：第二行三个图形的交点数均为3个，符合规律。
```

**【例3】**
```
题目：图形中的"死角"（线条端点）数量依次是（ ）。

选项：A. 2、3、4、5  B. 3、4、5、6  C. 4、5、6、7  D. 5、6、7、8

答案：B
解析：观察可知图形中线条端点数量分别为3、4、5、6。
```

**【例4】**
```
题目：下列图形中，含有两个交点的是（ ）。

选项：A. T形  B. L形  C.十字形  D.三角形

答案：A
解析：T形有一个线条交点和一个端点，但通常"交点"指线条交叉点。T形有1个交点，但"死角"有2个。十字形有1个交点但4个端点，故选A。
```

**【例5】**
```
题目：图形笔画数与交点数量之和为（ ）。

选项：A. 恒定值  B. 递增  C. 递减  D. 无规律

答案：A
解析：每个图形的笔画数与交点数之和均为4。
```

**二、线**

**规律说明**：图形中线条的数量（直线、曲线、射线）存在某种数量关系。

**题目样例**：

**【例1】**
```
题目：每组图形中，曲线数量相同的是（ ）。

选项：A. 第一组  B. 第二组  C. 第三组  D. 三组都不相同

答案：B
解析：第二组三个图形均包含2条曲线。
```

**【例2】**
```
题目：图形中直线与曲线的数量差为（ ）。

选项：A. 0  B. 1  C. 2  D. 3

答案：A
解析：图形中直线数量均为3，曲线数量均为2，差值为1。但部分图形直线与曲线数量相等。
```

**【例3】**
```
题目：下列图形笔画数分别为（ ）。

选项：A. 1、2、3、4  B. 2、3、4、5  C. 3、4、5、6  D. 4、5、6、7

答案：B
解析：一笔画图形（如圆）笔画数为1，两笔画图形（如字母Z）笔画数为2，以此类推。
```

**【例4】**
```
题目：图形中线段的数量依次是（ ）。

选项：A. 3、4、5、6  B. 4、5、6、7  C. 5、6、7、8  D. 6、7、8、9

答案：A
解析：统计各图形中不封闭的线段数量，分别为3、4、5、6。
```

**【例5】**
```
题目：每行图形中，线条总数相同的是（ ）。

选项：A. 第一行  B. 第二行  C. 第三行  D. 三组都不相同

答案：D
解析：经统计，三行图形的线条总数分别为6、7、8，不完全相同。
```

**题目样例**：

```
题目：图形中封闭区域的数量依次是（ ）。

选项：A. 1、2、3、4  B. 2、3、4、5  C. 3、4、5、6  D. 4、5、6、7

答案：B
解析：图形中封闭区域数量分别为2、3、4、5。
```

**【例2】**
```
题目：每组图形中，封闭面数量相同的是（ ）。

选项：A. 第一组  B. 第二组  C. 第三组  D. 三组都不相同

答案：A
解析：第一组三个图形的封闭面数量均为2。
```

**【例3】**
```
题目：下列图形中，含有3个封闭面的是（ ）。

选项：A. 品字  B. 森字  C. 田字  D. 罒字

答案：C
解析：品字有3个封闭面，森字有4个，田字有5个（四个小口加外部大框），罒字有4个封闭面。
```

**【例4】**
```
题目：图形中空白部分与阴影部分的比例为（ ）。

选项：A. 1:1  B. 2:1  C. 3:1  D. 4:1

答案：A
解析：图形被等分为两个相等的区域，空白与阴影部分面积相等。
```

**【例5】**
```
题目：每行图形的封闭面数量之和为（ ）。

选项：A. 6  B. 9  C. 12  D. 15

答案：C
解析：经统计，每行图形的封闭面数量之和均为12。
```

```
题目：图形中直角的数量依次是（ ）。

选项：A. 1、2、3、4  B. 2、3、4、5  C. 3、4、5、6  D. 4、5、6、7

答案：A
```

**【例2】**
```
题目：每组图形中，锐角数量相同的是（ ）。

选项：A. 第一组  B. 第二组  C. 第三组  D. 三组都不相同

答案：B
解析：第二组三个图形均包含4个锐角。
```

**【例3】**
```
题目：下列图形中，含有2个直角的是（ ）。

选项：A. L形  B. T形  C. 十字形  D. Z形

答案：A
解析：L形有1个直角，T形有1个直角，十字形有4个直角，Z形有0个直角。故选L形，但需注意有些L形变体含有2个直角。
```

**【例4】**
```
题目：图形中角度数之和为（ ）。

选项：A. 180°  B. 360°  C. 540°  D. 720°

答案：B
解析：多边形内角和公式为(n-2)×180°，该图形内角和均为360°。
```

**【例5】**
```
题目：每行图形中，角的数量（不含端点处角度）依次是（ ）。

选项：A. 3、4、5、6  B. 4、5、6、7  C. 5、6、7、8  D. 6、7、8、9

答案：C
解析：统计各图形中小于180°的内角数量，分别为5、6、7、8。
```

**题目样例**：

**【例1】**
```
题目：图形中独立元素的数量依次是（ ）。
```

**【例2】**
```

选项：A. 第一组  B. 第二组  C. 第三组  D. 三组都不相同

答案：A
解析：第一组三个图形均由2种不同元素组成。
```

**【例3】**
```
题目：下列图形中，含有3个三角形的是（ ）。

选项：A. 扇形  B. 菱形  C. 箭头  D. 十字

答案：D
解析：十字形可以分解为4个直角三角形（或更多取决于分割方式），菱形可分解为2个三角形，箭头可分解为1个三角形和1个矩形，扇形不含三角形。
```

**【例4】**
```
题目：图形中重复元素的数量为（ ）。

选项：A. 1个  B. 2个  C. 3个  D. 4个

答案：B
解析：图形中有2个相同的元素（如两个圆或两个三角形）。
```

**【例5】**
```
题目：每行图形中，元素笔画数之和依次是（ ）。

选项：A. 6、8、10、12  B. 7、9、11、13  C. 8、10、12、14  D. 9、11、13、15

答案：A
解析：统计各图形组成元素的笔画数之和，分别为6、8、10、12。
```

**六、综合应用**

**题目样例**：

**【例1】**
```
题目：观察下列图形规律，问号处应填（ ）。

选项：A. 5个点、3条线  B. 4个点、4条线  C. 3个点、5条线  D. 2个点、6条线

答案：B
解析：图形规律为"点数+线数=7"，且点数依次为3、4、5、？，线数依次为4、3、2、？，故问号处应为4个点、4条线。
```

**【例2】**
```
题目：每组图形中，面的数量比线的数量多（ ）个。

选项：A. 1  B. 2  C. 3  D. 0

答案：A
解析：第一组：面2-线1=1；第二组：面3-线2=1；第三组：面4-线3=1。
```

**【例3】**
```
题目：图形中点、线、面、角的数量关系为（ ）。

选项：A. 点=线  B. 线=面  C. 面=角  D. 点+面=线+角

答案：D
解析：经统计，每个图形均满足"点数+面数=线数+角数"。
```

#### 1.3.2 定义判断

**解题技巧**：
- **抓关键词**：主体、客体、条件、结果
- **排除法**：逐一排除不符合定义的选项
- **对比法**：比较选项与定义的匹配程度

#### 1.3.3 类比推理

**常见关系**：

| 关系类型 | 示例 |
|----------|------|
| 全同关系 | 番茄：西红柿 |
| 并列关系 | 苹果：香蕉 |
| 包含关系 | 水果：苹果 |
| 交叉关系 | 学生：党员 |
| 对应关系 | 医生：医院 |
| 因果关系 | 下雨：地湿 |
| 象征关系 | 鸽子：和平 |

#### 1.3.4 逻辑判断

**常见题型**：

1. **翻译推理**：
   - 如果A，那么B → A → B
   - 只有A，才B → B → A
   - 除非A，否则B → ¬A → B

2. **真假推理**：
   - 矛盾关系：必有一真一假
   - 反对关系：可以同真或同假

3. **削弱加强**：
   - 削弱方式：否定论点、否定论据、拆桥
   - 加强方式：肯定论点、补充论据、搭桥

4. **分析推理**：
   - 排除法：根据条件排除选项
   - 图表法：列表或画图辅助分析

### 1.4 资料分析

#### 1.4.1 统计术语

**基础术语**：
- **基期**：作为比较基准的时期
- **现期**：当前研究的时期
- **增长率**：(现期 - 基期) / 基期 × 100%
- **同比增长**：与上年同期相比
- **环比增长**：与上一周期相比
- **比重**：部分占总体的比例
- **平均数**：总数 ÷ 个数

#### 1.4.2 速算技巧

| 技巧 | 适用场景 | 示例 |
|------|---------|------|
| 估算法 | 选项差距较大 | 3456 ÷ 123 ≈ 28 |
| 直除法 | 分数比较 | 比较 5/7 和 3/4 |
| 截位法 | 数据较大时 | 34567 ÷ 1234 ≈ 346 ÷ 12 |
| 化同法 | 分数比较 | 将分母化为相同 |
| 差分法 | 分数比较 | 比较接近的分数 |

#### 1.4.3 常见陷阱

- **时间陷阱**：同比/环比、时间段范围
- **单位陷阱**：亿/万、百分比/百分点
- **概念陷阱**：增长率/增长量、比重/比例
- **范围陷阱**：全国/省份、累计/当月

## 2. 申论

### 2.1 归纳概括题

**题型特点**：
- 要求概括材料内容
- 字数限制严格（150-300字）
- 考察归纳提炼能力

**答题步骤**：
1. **通读材料**：了解整体内容
2. **定位关键信息**：圈画关键词句
3. **分类归纳**：按逻辑整理要点
4. **简洁表达**：用规范语言表述

**评分标准**：
- 要点全面（50%）
- 条理清晰（30%）
- 语言规范（20%）

### 2.2 综合分析题

**题型特点**：
- 要求分析现象、观点或问题
- 考察辩证思维能力
- 需要结合材料和实际

**常见类型**：
1. **观点分析**：评价某种观点的合理性
2. **现象分析**：分析现象产生的原因和影响
3. **关系分析**：分析两个事物之间的关系

**答题框架**：
1. **表明观点**：明确立场
2. **分析论证**：从不同角度分析
3. **总结提升**：给出结论或建议

### 2.3 提出对策题

**题型特点**：
- 要求针对问题提出解决方案
- 考察解决实际问题的能力
- 对策需具有可行性和针对性

**答题步骤**：
1. **分析问题**：明确问题本质
2. **确定对策方向**：从政策、管理、技术等角度
3. **具体措施**：分点列出可操作的措施
4. **评估可行性**：考虑实施条件和效果

**对策来源**：
- 材料中的现成对策
- 根据问题推导对策
- 借鉴实际经验

### 2.4 应用文写作

**常见文体**：

| 文体 | 特点 | 格式要求 |
|------|------|---------|
| 报告 | 陈述事实、汇报工作 | 标题、称谓、正文、落款 |
| 请示 | 请求上级指示批准 | 标题、主送机关、正文、请示语 |
| 通知 | 传达事项、要求办理 | 标题、主送机关、正文、落款 |
| 函 | 平行机关商洽工作 | 标题、主送机关、正文、结尾语 |
| 讲话稿 | 会议发言材料 | 标题、称谓、开场白、主体、结尾 |

**写作要求**：
- **格式正确**：符合文体规范
- **内容完整**：覆盖所有要点
- **语言得体**：符合身份和场合

### 2.5 文章论述题

**题型特点**：
- 要求写一篇议论文
- 字数要求：800-1200字
- 考察综合写作能力

**写作结构**：
1. **标题**：简明扼要，点明主旨
2. **开头**：引出话题，提出论点
3. **主体**：分论点论证（3-4个分论点）
4. **结尾**：总结全文，升华主题

**论证方法**：
- 举例论证：用具体事例证明观点
- 道理论证：引用名言、原理
- 对比论证：正反对比增强说服力
- 因果论证：分析原因和结果

**高分技巧**：
- **立意深刻**：观点新颖有深度
- **结构清晰**：逻辑严谨层次分明
- **论证充分**：论据充实有说服力
- **语言流畅**：表达准确生动

## 3. 面试

### 3.1 结构化面试

**常见题型**：

| 题型 | 考察能力 | 示例 |
|------|---------|------|
| 自我认知 | 自我定位、职业规划 | 请介绍一下你自己 |
| 综合分析 | 分析判断能力 | 如何看待互联网+政务？ |
| 人际沟通 | 沟通协调能力 | 同事意见分歧怎么办？ |
| 组织管理 | 计划组织能力 | 组织一次培训活动 |
| 应急应变 | 应变能力 | 突发公共事件如何处理？ |
| 专业知识 | 专业素养 | 谈谈对依法行政的理解 |

**答题思路**：
1. **自我认知**：突出优势、结合岗位
2. **综合分析**：是什么-为什么-怎么办
3. **人际沟通**：反思-沟通-解决-总结
4. **组织管理**：计划-准备-实施-总结
5. **应急应变**：冷静-处理-善后-总结

**面试礼仪**：
- **着装得体**：正式职业装
- **仪态大方**：坐姿端正、眼神交流
- **表达清晰**：语速适中、逻辑清晰
- **态度诚恳**：实事求是、不卑不亢

### 3.2 无领导小组讨论

**流程**：
1. **阅读材料**：5-10分钟
2. **个人陈述**：每人1-2分钟
3. **自由讨论**：20-30分钟
4. **总结陈词**：推荐代表总结

**角色定位**：
- **领导者**：引导讨论方向
- **组织者**：协调时间和流程
- **贡献者**：提供有价值观点
- **记录者**：记录讨论要点
- **总结者**：归纳总结讨论成果

**评分要点**：
- **内容贡献**：观点质量和数量
- **团队协作**：沟通和协调能力
- **领导能力**：影响力和组织能力
- **表达能力**：语言表达和逻辑思维

## 4. 公共基础知识

### 4.1 政治常识

#### 4.1.1 马克思主义基本原理

**核心内容**：
- 唯物论：物质与意识的关系
- 辩证法：矛盾分析法、发展观
- 认识论：实践与认识的关系
- 历史观：社会存在与社会意识

#### 4.1.2 毛泽东思想

**核心内容**：
- 新民主主义革命理论
- 社会主义改造理论
- 社会主义建设理论
- 党的建设理论

#### 4.1.3 中国特色社会主义理论体系

**核心内容**：
- 邓小平理论：改革开放、社会主义本质
- "三个代表"重要思想：代表先进生产力
- 科学发展观：以人为本、全面协调可持续
- 习近平新时代中国特色社会主义思想：新时代、新矛盾、新思想

### 4.2 法律常识

#### 4.2.1 宪法

**重点内容**：
- 国家性质：人民民主专政
- 根本制度：社会主义制度
- 公民权利：平等权、政治权利、人身自由
- 国家机构：全国人大、国务院、司法机关

#### 4.2.2 行政法

**重点内容**：
- 行政许可：许可设定、实施程序
- 行政处罚：种类、程序、听证
- 行政复议：范围、程序、决定
- 行政诉讼：受案范围、举证责任

#### 4.2.3 民法

**重点内容**：
- 民法总则：基本原则、民事主体
- 物权法：所有权、用益物权、担保物权
- 合同法：合同订立、效力、履行
- 侵权责任：归责原则、责任承担

### 4.3 经济常识

#### 4.3.1 宏观经济

**核心指标**：
- GDP：国内生产总值
- GNP：国民生产总值
- CPI：居民消费价格指数
- PPI：生产者价格指数
- 失业率：失业人口比例

**宏观政策**：
- 财政政策：税收、财政支出
- 货币政策：利率、存款准备金率、公开市场操作

#### 4.3.2 微观经济

**市场类型**：
- 完全竞争市场：农产品市场
- 垄断竞争市场：零售业
- 寡头垄断市场：汽车制造业
- 完全垄断市场：公用事业

### 4.4 管理常识

#### 4.4.1 行政管理

**核心职能**：
- 计划：制定目标和方案
- 组织：配置资源和人员
- 领导：激励和引导下属
- 控制：监督和评估绩效

**管理原理**：
- 系统原理：整体观念
- 人本原理：以人为本
- 责任原理：明确责任
- 效益原理：追求效益

#### 4.4.2 公共管理

**核心内容**：
- 公共政策：制定、执行、评估
- 公共服务：提供公共产品
- 政府职能：经济调节、市场监管、社会管理、公共服务

### 4.5 公文写作

#### 4.5.1 公文格式

**组成部分**：
- 版头：份号、密级、紧急程度、发文机关标志、发文字号
