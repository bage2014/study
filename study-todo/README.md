1. **跳表的本质是一个多层的有序链表，同时结合了二分和链表的思想；**
2. 由很多层索引组成，每一层索引是通过**随机函数**随机产生的，每一层都是一个有序的链表，默认是升序 ；
3. 最底层的链表包含所有元素；
4. 如果一个元素出现在第i层的链表中，则它在第i层之下的链表也都会出现；
5. 跳表的每个节点包含两个指针，一个指向同一链表中的下一个元素，一个指向下面一层的元素。
